# E-Handel-Admin

## Beskrivning
En enkel e-handelsapplikation byggd i C#/.NET med SQLite och Entity Framework Core.
Applikationen är en konsolapp som hanterar kategorier, produkter, kunder och ordrar och orderrow.

## Teknik
- .NET
- C#
- Entity Framework Core
- SQLite

## Kör projektet
1. Klona repot
2. Öppna lösningen i visual Studio
3. Kör projektet
4. Databasen skapas automatiskt via migrationer och seed-data

## Funktioner
- Crud för Category och Produkt
- Ordrar och orderrader
- Hash + salt för lösenord
- kryptering av e-postadress
